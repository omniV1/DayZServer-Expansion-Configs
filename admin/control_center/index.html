What this app does

This is a safe local control panel for your DayZ server folder. It reads map status, runs known admin scripts, captures output, and creates snapshots before guarded writes. It does not expose a public web server and it does not accept arbitrary shell commands.

1

Confirm the install

Use Setup Helper first when a server was freshly cloned, moved, or shared with another admin.

2

Pick a map

Dashboard cards show configs, missions, ports, mods, and whether the latest log reached ready state.

3

Diagnose before changing

Run config drift, latest-log triage, or LAN visibility checks before touching generation tools.

4

Tune carefully

Balance saves create snapshots. Loot preset changes still need Apply Loot Now, and most AI/zombie edits need a restart.

Safe Daily Workflow

  1. Refresh the dashboard.
  2. Check the map you want to work on.
  3. Run Config Drift or Triage Logs if anything looks off.
  4. Create a snapshot before larger changes.
  5. Apply changes, restart the affected map, then smoke test.

Risk Labels

read
Looks only. Safe anytime.
guarded
Writes files or starts/stops a test process, usually after a snapshot.
high
Can stop servers, wipe imported storage, or rebuild generated configs. Requires typed confirmation.

Terms

A private backup zip of configs before risky work.

World Events

Tune vehicles, helicopter crashes, airdrops, and static events for one map. nominal is the target count, min/max bound it, and lifetime is how long (seconds) an event persists. Saving previews the exact change first, snapshots, then writes db/events.xml. Restart the map afterward.
Select a map to load its world events.

Mission Builder

Create repeatable paid contracts (Hryvnia) for any map without hand-editing quest JSON. Missions install into the map's private profile and use the existing contract board. Preview shows every file before anything is written, snapshots first, and never overwrites. Restart the map after creating a mission.

Support Report

Generate a public-safe summary you can paste into a support thread or Discord. It lists app version, configs, ports, missions, missing mods, and the latest log state for one map or all maps. Passwords, Steam IDs, and tokens are redacted, and no player, storage, or profile contents are included. 
Choose a scope and select Generate Report.

Maintenance

Checks and guarded repair actions
Start here when something feels broken. Read-only diagnostics are safe while servers are running. Guarded actions may write files or sync profiles and will say so before they run.

Fix Problems
