## Capture tips

- Run `python admin\control_center.py --open-browser` (or the EXE) and use your browser at ~1280-1440px wide.
- **Dark mode** screenshots tend to look the most "pro" — toggle Theme → Dark in the header.
- Keep windows tidy; crop to the app content (no taskbar/desktop).
- PNG for static shots, GIF for the demo (keep GIFs under ~10 MB so GitHub renders them inline).
- Nothing private is shown by the app, but double-check no real Steam IDs/IPs are visible before committing.
